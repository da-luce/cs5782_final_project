# LoRA Re-implementation (CS 4/5782 Final Project)

## 1. Introduction

This repository contains our re-implementation of *LoRA: Low-Rank Adaptation of Large Language Models*. We study whether LoRA can match full fine-tuning performance while using far fewer trainable parameters.

## 2. Chosen Result

We reproduce results comparing LoRA vs full fine-tuning on RoBERTa-base for:

* SST-2 (sentiment classification)
* MNLI (natural language inference)

Goal: Compare accuracy and parameter efficiency.
